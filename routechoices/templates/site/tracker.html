GPS Trackers

Sharing your position

Sharing your position to provide live GPS tracking can be done in different ways. Using our smartphone application, a modern Garmin watch loaded with our data field, or using one of the supported dedicated GPS trackers.

In any case, the device you are using will be assigned a unique Device ID that is used to identify your device and that you can communicate to the events organisers so you can be tracked.

Specific Instructions

Install the app from Google Play or the Apple App Store
Search "Routechoices Tracker", by OneHertz.

The Routechoices Tracker App lets you stream your live location to Routechoices.com. It provides you with a unique device ID, which you can share with event organizers to be registered.

How It Works:

  1. Open the app.
  2. Grant it access to high accuracy location at all time, as well as background execution permission.
    This ensures uninterrupted GPS tracking throughout your event..
  3. Press "Start Live GPS" when you're ready to begin your run. The Device ID turns green when you are "Live".
  4. Go on your adventure!
  5. Press "Stop Live GPS" when you've finished.

Troubleshooting:

  • Ensure that high-accuracy GPS is enabled in your phone's system settings.
  • Recent Android versions may restrict background processes to improve battery life, which can affect apps like Routechoices Tracker.
    To ensure continuous tracking, add Routechoices Tracker to battery optimization exceptions in your device settings.
  • Some Android manufacturers have additional battery-saving settingsâ€”make sure to disable them or add the app to their exceptions list.
  • iOS restricts applications that can run in the background, make sure that background execution is allowed and Routechoices Tracker has permission to access location services at all times.
  • iOS also limits when an application can fetch location data, ensure that the privacy setting for location access in "Settings > Privacy > Location Services > Routechoices Tracker" is set to "Always"

Install the data field for Garmin watches from the Garmin Connect IQ App Store
Search "Routechoices Tracker"

How It Works:

Once installed, add the data field to the activity profiles in which you want to be tracked.

You'll need a connection to a phone with bluetooth during the activity to be able send locations to the server.

The field displays a Device ID you can use to register to events and the status of connection with the server (OFFLINE/LIVE)

Supported trackers

The dedicated GPS trackers currently supported are:

  • GT06 protocol devices
  • Mictrack MT710
  • Queclink devices
  • Teltonika TMT250
  • Tracktape devices
  • Xexun protocol devices
  • Xexun2 protocol devices

Own a product not listed above? Enquire us about adding support for its protocol!

Device's configuration

Configure the devices to share data over TCP with the server at address routechoices.com on the following port:

  • 5023: GT06
  • 5191: Mictrack
  • 5004: Queclink
  • 5027: Teltonika
  • 2003: TrackTape
  • 5006: Xexun
  • 5233: Xexun2
The recommended data push interval is 5 seconds, and the recommended GPS data acquisition interval is 1 second, but feel free to adjust the settings depending on your usage as this settings affect the battery life.
