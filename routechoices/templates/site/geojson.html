Routechoices.com GeoJSON CSS Guide

GeoJSON allows you to draw features on your map.

You can combine multiple lines and points under the features element of the JSON.

Add the style informations you want for each feature.

Find more info at GeoJSON.io

Quick Examples

Polygons

{
  "type": "FeatureCollection",
  "features": [
    {
      "type": "Feature",
      "properties": {},
      "style": {
        "color": "#CC0000",
        "weight": 2,
        "opacity": 1,
        "dashArray": "3, 5",
        "fillColor": "#0000CC",
        "fill-opacity": 0.6
      },
      "geometry": {
        "type": "Polygon",
        "coordinates": [[
          [37.61489152908325,55.752586015356876],
          [37.61577129364014,55.7539867694403],
          [37.616543769836426,55.75506145183324],
          [37.6177453994751,55.75443355110991],
          [37.619526386260986,55.753491681072205],
          [37.62143611907959,55.75252563689488],
          [37.62117862701416,55.75211506087468],
          [37.61875391006469,55.750762544596384],
          [37.61813163757324,55.749989657097],
          [37.61302471160889,55.749035602973365],
          [37.612552642822266,55.74907183330299],
          [37.6134967803955,55.750641781933986],
          [37.61489152908325,55.752586015356876]
        ]]
      }
    }
  ]
}

Lines

{
  "type": "FeatureCollection",
  "features": [
    {
      "type": "Feature",
      "properties": {},
      "style": {
        "color": "#0099ff",
        "weight": 9,
        "opacity": 0.7,
        "className": "stroke-light"
      },
      "geometry": {
        "type": "LineString",
        "coordinates": [
          [4.364140,50.84248],
          [4.564510,50.84241],
          [4.464510,51.04241],
        ]
      }
    }
  ]
}

Points

{
  "type": "FeatureCollection",
  "features": [
    {
      "type": "Feature",
      "properties": {},
      "style": {
        "icon": {
          "className": "map-marker start-icon"
        }
      },
      "geometry": {
        "type": "Point",
        "coordinates": [4.36414,50.84248]
      }
    }
  ]
}

"style" > "icon" > "className" can be set to one of the following:

  • map-marker start-icon
  • map-marker end-icon
  • map-marker waypoint
  • map-marker waypoint grey
  • map-marker waypoint red
  • map-marker waypoint orange
  • map-marker waypoint green
  • map-marker waypoint dark-green
  • map-marker waypoint dark-green
  • distance-marker warning
  • distance-marker red
  • distance-marker green
  • map-marker cat1
  • map-marker cat2
  • map-marker cat3
  • map-marker cat4
  • map-marker cathc
  • distance-marker d1
  • distance-marker d2
  • distance-marker d3
  • ...
  • distance-marker d9
  • distance-marker d10
  • distance-marker d15
  • distance-marker d20
  • ...
  • distance-marker d95
  • distance-marker d100
  • distance-marker d110
  • distance-marker d120
  • ...
  • distance-marker d490
  • distance-marker d500

Or you can also use any of the FontAwesome 6.7.2 icons by using the "leaflet-fa-marker" class name in addition of the fontawesome class of the icon you want. (e.g. "leaflet-fa-marker fa-solid fa-map-pin")

You can also add a label to the point, with adding a "properties" > "text" value
